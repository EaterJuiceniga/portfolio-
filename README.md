# Business Card Generator

A simple React.js app for generating professional business cards. Users can input their information and see a live preview of their custom business card.

## Features

- Live preview as you type
- Customizable name, title, company, email, phone, and website
- Clean and modern UI

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Run the app locally

```bash
npm start
```

### 3. Build for production

```bash
npm run build
```
